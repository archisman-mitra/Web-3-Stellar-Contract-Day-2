# Web-3-Stellar-Contract-Day-2

🚀 **Web3 RiseIn Workshop – Will and Testament Contract (Day 2)**

A **Soroban-based Web3 application** built during the RiseIn Workshop. This project demonstrates how to create, deploy, and interact with a **Will and Testament smart contract** on the Stellar network through a simple frontend interface.

---

## 📌 Project Overview

This project demonstrates a complete **Web3 dApp workflow**, including:

- 🧠 Smart contract development using **Soroban**
- 🔗 Contract deployment on the **Stellar network**
- 💻 Frontend interface for contract interaction
- 🔐 Wallet integration using **Freighter**

---

## 📜 Smart Contract

The core logic of this project is implemented as a **Will and Testament smart contract** using Soroban.

This contract is designed to demonstrate blockchain-based inheritance or asset transfer logic in a decentralized way.

### Key Highlights
- 📄 Will and testament contract logic
- 🔐 Secure and decentralized execution
- ⚙️ Smart contract deployment on Stellar
